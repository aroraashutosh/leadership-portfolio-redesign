# Leadership Portfolio Redesign

Assignment submission for the Product Management Internship at DeepThought.

## What this is
A redesign of the PDGMS Leadership Portfolio page — transforming it from a
dashboard-style layout into a narrative-first "career pitch deck," using
real mock data for one employee (Rajan Iyer, March 2026).

## Files
- `leadership-portfolio.html` — working prototype (open in any browser)
- `design-rationale.pdf` — write-up of design decisions and tradeoffs
